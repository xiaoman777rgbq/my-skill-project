📚 变量定义

1. 使用 var 声明变量（ES5）

var 是函数作用域，存在变量提升

var name = "小明";
var age = 18;
var isStudent = true;
console.log(name, age, isStudent);

2. 使用 let 声明变量（ES6）

let 是块级作用域，不能重复声明

let score = 95;
let grade = 'A';
console.log("成绩:", score, "等级:", grade);

3. 使用 const 声明常量（ES6）

const 声明后不能重新赋值，且必须初始化

const PI = 3.14159;
const APP_NAME = "我的应用";
// PI = 3.14; // 错误！不能重新赋值

4. 变量命名规则

// 合法命名
var userName = "张三";      // 驼峰命名
var $price = 99.9;          // 可以用$
var _private = "私有";      // 可以用_
var user2 = "李四";         // 数字不能在开头

// 不合法
// var 2name = "错误";      // 不能以数字开头
// var my-name = "错误";    // 不能用中划线

📝 练习题

练习 1：交换两个变量的值

题目：不使用第三个变量，交换 a 和 b 的值

// 初始值
var a = 10;
var b = 20;

// 交换后：a = 20, b = 10
💡 答案提示（点击展开）

解题思路：使用算术运算或解构赋值

// 方法1：算术运算
a = a + b;  // a = 30
b = a - b;  // b = 10
a = a - b;  // a = 20

// 方法2：解构赋值（ES6）
[a, b] = [b, a];

练习 2：判断常量是否可以修改

题目：const 声明的对象属性可以修改吗？数组元素呢？

const person = { name: "小明", age: 20 };
const fruits = ["苹果", "香蕉"];

// 能否执行以下操作？
// person.age = 21;      // ❓
// fruits.push("橙子");   // ❓
💡 答案提示（点击展开）

答案：都可以修改！const 保证的是引用不变，不是内容不变

const person = { name: "小明", age: 20 };
person.age = 21;        // ✅ 可以
person.city = "北京";   // ✅ 可以添加新属性

const fruits = ["苹果", "香蕉"];
fruits.push("橙子");    // ✅ 可以
fruits = ["葡萄"];      // ❌ 错误！不能重新赋值
