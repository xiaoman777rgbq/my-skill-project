📚 变量定义

1. 使用 var 声明变量（ES5）

var 是函数作用域，存在变量提升

var name = "小明";
var age = 18;
var isStudent = true;
console.log(name, age, isStudent);

2. 使用 let 声明变量（ES6）

let 是块级作用域，不能重复声明

let score = 95;
let grade = 'A';
console.log("成绩:", score, "等级:", grade);

3. 使用 const 声明常量（ES6）

const 声明后不能重新赋值，且必须初始化

const PI = 3.14159;
const APP_NAME = "我的应用";
// PI = 3.14; // 错误！不能重新赋值

4. 变量命名规则

// 合法命名
var userName = "张三";      // 驼峰命名
var $price = 99.9;          // 可以用$
var _private = "私有";      // 可以用_
var user2 = "李四";         // 数字不能在开头

// 不合法
// var 2name = "错误";      // 不能以数字开头
// var my-name = "错误";    // 不能用中划线
