📚 JavaScript 变量类型

1. 字符串 String

var name = "你好世界";
var greeting = 'Hello';

2. 数字 Number

var age = 25;
var price = 99.9;

3. 布尔值 Boolean

var isActive = true;
var isEmpty = false;

4. 未定义 Undefined

var notAssigned;
console.log(notAssigned);

5. 空值 Null

var empty = null;
console.log(empty);

6. 数组 Array

var fruits = ["苹果", "香蕉"];

7. 对象 Object

var person = {
  name: "小明",
  age: 20
};

8. typeof 检测类型

typeof "hello"  // "string"
typeof 123      // "number"
typeof true     // "boolean"

💡 类型转换示例

// 字符串转数字
var num = Number("123");     // 123
var nan = Number("abc");      // NaN

// 数字转字符串
var str = String(123);       // "123"
var str2 = (123).toString(); // "123"

// 强制类型转换
var bool = Boolean(1);       // true
var bool2 = Boolean(0);      // false

📝 练习题

练习 1：类型判断挑战

题目：以下表达式的 typeof 结果是什么？

typeof "123"
typeof 123
typeof true
typeof undefined
typeof null        // ⚠️ 经典坑！
typeof [1, 2, 3]   // ⚠️ 经典坑！
typeof {name: "Tom"}
typeof function(){}
💡 答案提示（点击展开） 
typeof "123"           // "string"
typeof 123             // "number"
typeof true            // "boolean"
typeof undefined       // "undefined"
typeof null            // "object"      ← 经典坑！
typeof [1, 2, 3]       // "object"     ← 数组也是对象！
typeof {name: "Tom"}   // "object"
typeof function(){}    // "function"

面试重点：为什么 null 的 typeof 是 "object"？
这是 JavaScript 早期的 bug，但为了兼容性一直保留至今。

练习 2：类型转换陷阱

题目：计算以下表达式的结果

"5" + 3        // ⚠️ 
"5" - 3         // ⚠️
"5" * "2"       // ⚠️
"hello" - 1     // ⚠️
true + 1        // ⚠️
false + "1"     // ⚠️
💡 答案提示（点击展开）

规律：+ 运算符会拼接字符串，其他运算符会转为数字

"5" + 3    → "5" + "3" = "53"  (字符串拼接)
"5" - 3    → 5 - 3 = 2         (减法转数字)
"5" * "2"  → 5 * 2 = 10        (乘法转数字)
"hello" - 1 → NaN             (无法转数字)
true + 1   → 1 + 1 = 2         (true转数字1)
false + "1" → "false1"         (字符串拼接)
